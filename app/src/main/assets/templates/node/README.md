# Apex Node.js 项目

这是一个使用 Apex 创建的 Node.js 项目。

## 快速开始

1. 点击 "npm install" 安装依赖
2. 点击 "npm start" 启动服务器
3. 点击"浏览器预览"查看运行效果

## 项目结构

- `index.js` - 主程序入口
- `package.json` - 项目配置文件
- `.Apex/config.json` - Apex 工作区配置

## 自定义

您可以：
- 修改 `index.js` 添加您的业务逻辑
- 在 `package.json` 中添加依赖包
- 在 `.Apex/config.json` 中自定义命令按钮
