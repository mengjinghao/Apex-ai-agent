rocket_launch 快速开始

1 下载安装

从官网下载最新的 APK 文件并安装到您的 Android 设备。

终端 
# 下载 APK
adb install apex-agent.apk

# 启动应用
adb shell am start -n com.apex.agent/.MainActivity

2 授予权限

首次启动时，请授予以下必要权限以确保应用正常运行。

accessibility 无障碍权限
folder 存储权限
wifi 网络权限
notifications 通知权限

3 配置 API

在设置中配置您的 AI 模型 API 密钥以启用高级功能。

config.json 
{
  "api_base_url": "https://api.example.com",
  "api_key": "your-api-key-here",
  "model": "gpt-4",
  "max_tokens": 2048
}

auto_awesome 核心功能

狂暴模式是 Apex-Agent 的核心功能，通过智能 CPU 调度和任务优化，实现高达 4 倍的任务处理速度提升。

  • 智能断点续传机制
  • 批量任务并行处理
  • 自动错误恢复与重试
  • 实时进度监控

多个专用 AI 代理协同工作，自动分解复杂任务并智能汇总结果。

  • 主代理：任务调度与协调
  • 数据代理：信息处理与分析
  • 网络代理：网页交互与采集
  • 汇总代理：结果整合与输出

多层记忆系统让 AI 真正理解用户，提供个性化的智能体验。

  • 短期记忆：实时对话上下文
  • 长期记忆：用户偏好与习惯
  • 知识图谱：概念关联与语义理解

help 常见问题

help 支持哪些 Android 版本？

Apex-Agent 支持 Android 7.0 (API 24) 及以上版本，包括最新的 Android 14。

help 如何获取 API 密钥？

您可以在 OpenAI、Claude 或其他 AI 服务提供商的官网申请 API 密钥。 Apex-Agent 支持多种 AI 模型。

help 我的数据是否安全？

完全安全！所有数据处理都在您的设备本地完成，不会上传到任何服务器。我们重视您的隐私保护。

help 应用耗电吗？

Apex-Agent 采用了智能电源管理策略，在非活跃期间自动降低资源占用。狂暴模式虽然会提高性能，但也会注意控制功耗。
