# Apex-Agent 下载页面 (download.html) 实施计划

## 概述
基于现有 `index.html` 的设计风格和双主题系统，创建完整的 `download.html` 下载页面。

## 页面结构

### 1. 头部 (Head)
- 与 index.html 一致的元数据、字体、图标引入
- 相同的 Tailwind CSS 配置和自定义样式
- 页面标题设为 "下载 | Apex-Agent"

### 2. 背景元素
- 动态背景球体 (bg-orbs)
- 液态玻璃背景 (liquid-bg)
- 网格覆盖层 (grid-overlay)
- 数字雨画布 (digitalRain)
- 像素网格和扫描线效果

### 3. 导航栏 (Header)
- 与主页完全一致的导航栏
- Logo 链接到 index.html
- 导航链接：首页、功能、文档、关于
- 主题切换按钮（霓虹东京/液态玻璃）
- 移动端菜单

### 4. 主内容区域 (Main)

#### 4.1 页面标题和 Hero 区域
- 标题："下载 Apex-Agent"
- 副标题："获取最新版本，体验移动端AI自动化"
- 视觉元素和动画效果

#### 4.2 下载区域
- 最新版本 APK 下载按钮（主 CTA）
- 版本信息展示（v2.0.0、发布日期、文件大小）
- 备用下载链接
- SHA256 校验信息

#### 4.3 系统要求说明
- Android 版本要求
- 内存要求
- 存储空间要求
- 可选权限说明

#### 4.4 版本历史/更新日志
- 按时间倒序排列的版本记录
- 每个版本的更新内容
- 下载链接和日期

#### 4.5 安全提示
- 应用安全说明
- 权限透明性
- 数据隐私保护

### 5. 页脚 (Footer)
- 与主页完全一致的 Footer
- 导航链接、社交媒体图标等

### 6. 返回顶部按钮
- 与主页一致的返回顶部功能

### 7. JavaScript 功能
- 双主题系统（霓虹东京/液态玻璃）
- 数字雨效果
- 液态背景追踪
- 卡片光泽效果
- 移动端菜单
- 返回顶部
- 卡片倾斜效果

## 技术栈
- HTML5
- Tailwind CSS（带自定义配置）
- Material Icons
- 原生 JavaScript（无外部库依赖）

## 设计规范
- 与 index.html 保持一致的视觉风格
- 响应式设计，支持移动端
- 完整的双主题支持
- 遵循 Material Design 3 风格
- 平滑的动画和交互效果
